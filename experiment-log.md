## Итерации Codex

### Итерация 1

- Промпт: создать учебный проект с Node.js CLI для Handlebars, тестами, README, отчетом, журналом и презентацией.
- Результат: создана структура проекта, исходный код CLI, тесты, примеры данных и документация.
- Ошибки: первый запуск `npm test` показал, что Vitest нельзя импортировать через `require()` в CommonJS-тесте.
- Исправления: тестовый файл переведен в ESM-формат `.mjs`, сам CLI оставлен на CommonJS.
- Итоговая проверка: `npm install`, `npm test`, запуск примера.

## Промпты

Основной промпт:

```text
Создать Node.js CLI-инструмент для генерации HTML/текста из Handlebars-шаблона и JSON-контекста, добавить тесты Vitest, примеры, README, REPORT, experiment-log и презентацию. Описать процесс генерации и проверки проекта с помощью Codex.
```

## Ошибки и исправления

| Ошибка | Причина | Исправление | Статус |
| --- | --- | --- | --- |
| Невалидный JSON в `data/broken-context.json` | Демонстрационный сценарий ошибки | Оставлен намеренно | Ожидаемое поведение |
| `Vitest cannot be imported in a CommonJS module using require()` | Vitest v1 ожидает ESM-импорт API тестов | `test/renderer.test.js` заменен на `test/renderer.test.mjs` с `import` | Исправлено |

## Итоговые результаты

- CLI принимает `--template`, `--context`, `--out`.
- При отсутствии `--out` результат выводится в stdout.
- Ошибки ввода и рендеринга обрабатываются.
- Тесты проверяют простой шаблон, вложенные поля, `each`, невалидный JSON и запись в файл.
- Результаты описывают генерацию и проверку проекта с помощью Codex.
